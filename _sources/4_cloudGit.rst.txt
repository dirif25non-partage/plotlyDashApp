Travail avec Google Cloud Shell & GitHub
#############################################
Avec le PC Pro, il n'est pas possible d'utiliser VS Code et Git. Un alternative est 
de travailler dans le Cloud Shell de Google et de synchroniser le travail sur GitHub.

Le plus facile pour se connecter à Google est d'utiliser des outils intégrés de Cloud Code qui rendent l'authentification plus simple.

Pour synchroniser le Cloud Shell avec GitHub, il est aussi possible d'utiliser un
`tocken de GitHub. <https://docs.github.com/en/authentication/keeping-your-account-and-data-secure/managing-your-personal-access-tokens>`_

`Pour générer un nouveau tocken <https://github.com/settings/tokens>`_
