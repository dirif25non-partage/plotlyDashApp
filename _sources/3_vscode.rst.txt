Utilisation de VS Code et Google Run
######################################
Lorsque l'on suit la documentation de Dash et que l'on veut déployer les applications 
qui sont fournies en exemple, il faut satisfaire plusieurs condditions pour 
que l'application fonctionne.

Dans un premier temps, on fait fonctionner le programme localement. 
L'application est lancée par les fonctions Debug de VS Code.
Il faut que le fichier actif soit app.py.

Je n'ai pas rencontré de defficulté à ce stade et le debbuggeur avait des messages 
suffisement explicites.
